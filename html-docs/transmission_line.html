Transmission line¶

Author: Michał Porębski

class transmission_line.TransmissionLine[source]¶

Main class which instance is optical transmission line through which we can pass rays, represented by list of optical elements. All lengths are in millimeters, angles usually can be put in degrees of radians

Example of use:

line = TransmissionLine([
    Air(length=50.8),
    ParabolicMirror(diameter=50.8, rfl=50.8, pfl=25.4, relative_angle_deg=0, mirrored=True),
    Air(length=70),
    ParabolicMirror(diameter=50.8, rfl=50.8, pfl=25.4, relative_angle_deg=0, mirrored=False),
    Air(length=2 * 50.8),
    ParabolicMirror(diameter=50.8, rfl=50.8, pfl=25.4, relative_angle_deg=0, mirrored=True),
    Air(length=90),
    PrismZnSe(length=50, construction_angle_deg=27, angle_to_beam_deg=17, thickness=40),
    Air(length=50.8),
    ParabolicMirror(diameter=50.8, rfl=50.8, pfl=25.4, relative_angle_deg=0, mirrored=False),
    Air(length=50.8),
    Screen(diameter=12.8, height=0.5, relative_angle_deg=-38 - 23)
])
line.update()
ax = line.plot(plot_type=settings.plot_type)
plt.show()
