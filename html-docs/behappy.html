HITRAN database access¶

Author: Natalia Stalmach Hitran database: https://hitran.org/

Air compounds: https://www.thoughtco.com/chemical-composition-of-air-604288

Humidity calculations: https://www.aqua-calc.com/calculate/humidity

class behapi.Behapi(temperature_celsius, pressure_pa, relative_humidity, atmosphere)[source]¶

Class Behapi consist of methods for using data from HITRAN database for calculating transmittance spectrum for object with defined temperature, pressure, relative humidity and gas mixture.

Example of use:

char = Behapi(temperature_celsius=20, pressure_pa=101325,
          relative_humidity=20, atmosphere='nitrogen')
char.absorption_coefficient(method='sdvoigt', wavelength_range_um=[0.5, 50], wavelength_step_um=0.00005)
# deswave = np.arange(2.5, 20, 17.5/2000)
deswave = np.arange(0.5, 50, 0.00005)
plt.figure()
plt.title(
    'Transmittance spectrum for nitrogen environment with 20°C temperature, 1013.25Pa pressure, 20% relative humidity and 30cm beam path',
    fontsize=15)
plt.xlabel('Wavelength [$\mu m$]', fontsize=13)
plt.ylabel('Transmittance', fontsize=13)
plt.plot(deswave, char.transmittance(distance_cm=30, desired_wavelengths=deswave))
plt.savefig('transmittance.svg')
absorption_coefficient(method, wavelength_range_um=[2.5, 20], wavelength_step_um=5e-05, wavelength_step_doppler_um=5e-06)[source]¶

Method for calculating absorption coefficient, having predefined lineshape (method) and desired wavelengths - converts given wavelength range and wavelength step to wavenumbers

Parameters:
  • method (string) – method of calculating transmittance spectrum based on taken lineshape - ‘voigt’, ‘doppler’ or ‘sdvoigt’ (required)
  • wavelength_range_um (list) – wavelength range in which transmittance spectrum should be calculated given in um, default is [2.5, 20] (optional)
  • wavelength_step (float) – wavelength step given in um, default is 0.00005 (optional)
  • wavelength_step_doppler (float) – wavelength step for Doppler method given in um, default is 0.000005 (optional)
absorption_coefficient_air(method, wavenumber_range, wavenumber_step, wavenumber_step_doppler)[source]¶

Auxiliary method for absorption_coefficient(), designed for objects with air atmosphere, based on wavenumber notation - updates database basing on the newest data form hitran, checks if every necessery data is available and if not downloads it - with given lineshape (method) calculates proper absorption coefficient

Parameters:
  • method (string) – method of calculating transmittance spectrum based on taken lineshape - ‘voigt’, ‘doppler’ or ‘sdvoigt’ (required)
  • wavenumber_range (list) – wavenumber range in which transmittance spectrum should be calculated given in cm-1 (optional)
  • wavenumber_step (float) – wavenumber step given in cm-1 (optional)
  • wavenumber_step_doppler (float) – wavenumber step for Doppler method given in cm-1 (optional)

transmittance(distance_cm, desired_wavelengths)[source]¶

Method returns transmittance spectrum for given environment in given wavelength range. Default values of wavelength step are chosen basing on hapi documentation (in wavenumber 0.01 and 0.001 respectively). It’s not recommended to go above those values because of possible accuracy decline. Maximum wavelength range is [0.5, 50] um because of database which is downloaded for this range - if needed it can be changed in the code.

Parameters:
  • distance_cm (float) – distance traveled by beam given in cm (required)
  • array desired_wavelengths (numpy) – numpy array of wavelengths which the user needs, given in um (required)
Return numpy array transmittance:
 

values of transmittance for desired wavelengths in numpy array
