tools_functions.line_intersection_with_paraboloid(par_coeff, start_p, dir, mirror_x_domain, mirror_y_domain, accuracy=1e-10)[source]¶

Find line intersection with paraboloid of center in (0,0), paraboloid constructed with equation:

y = (x**2 + z**2) * par_coeff

calculated intersection using analytical calculations, solves set of equations:

x(t) = x0 + t * a
y(t) = y0 + t * b
z(t) = z0 + t * c
y(t) = (x**2(t) + z**2(t)) * par_coeff
Parameters:
  • par_coeff (float) – parabola coefficient
  • start_p (np.array) – [:, [x,y,z]], line starting point, have to be 2d numpy array
  • dir (np.array) – [:, [x,y,z]], line direction vector
  • mirror_x_domain (list) – [min, max] domain where the mirror is defined in x axis
  • mirror_y_domain (list) – [min, max] domain where the mirror is defined in y axis
  • accuracy (float) – accuracy of computations, below it, values are treated as zeroes
Returns:

(np.array) intersection_point:
 vector with the same length as start_p with coordinates of intersection points
(np.array) normal_vectors:
 normal_vectors in this points

Find line intersection with paraboloid of center in 0,0, paraboloid constructed with equation:

y = (x**2 + z**2) * par_coeff

using analytical calculations, solves set of equations:

(x-x0)/a = (y-y0)/b = (z-z0)/c
y = (x**2 + z**2) * par_coeff
Parameters:

  • start_p (np.array) – [:, [x,y,z]], line starting point, have to be 2d numpy array
  • dir (np.array) – [:, [x,y,z]], line direction vector
  • mirror_x_domain (list) – [min, max] domain where the mirror is defined in x axis
  • mirror_y_domain (list) – [min, max] domain where the mirror is defined in y axis
  • accuracy (float) – accuracy of computations, below it, values are treated as zeroes
  • force_y_method (bool) – flag to force calculations using y variable as unknown
Returns:

(np.array) intersection_point:
 vector with the same length as start_p with coordinates of intersection points
