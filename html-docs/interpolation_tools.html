refle_interpolation.linear_interpolation(data, part_to_extrapolate=2)[source]¶

Linear interpolation with extrapolation, for extrapolation it uses only some part of characteristics, part_to_extrapolate determines the divider how much of lasts samples are taken into consideration, for example 2 means half, 5 means one fifth of last samples

Parameters:
  • data (np.array) – 2 column np.array
  • part_to_extrapolate (float) – divider which determines how much samples are taken into extrapolation
Returns:

(callable) interpolation
