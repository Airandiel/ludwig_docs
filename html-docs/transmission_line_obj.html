Optical elements¶

Author: Michał Porębski Transmission line is made of different components, in this module they are implemented:

  • Vacuum
  • Air
  • FlatMirror
  • ParabolicMirror
  • Window
  • Spherical lens
  • Screen
  • Prism
    • PrismZnSe

All components inherit from TransmissionObject class. New components can be easily added by overriding the methods of parent class. Main assumptions:

  • angle is angle of rotation of element
  • default paths to characteristics are defined in settings file
  • lengths are in millimeters

class transmission_line_obj.ParabolicMirror(relative_angle_deg: float = None, relative_angle: float = 0, rfl: float = None, pfl: float = None, mirrored: bool = False, **kwargs)[source]¶
Parabolic Mirrors:

needs reflected focal length, and parent focal length, -rfl - reflected focal length

-pfl - parent focal length

-relative_angle - can rotate component according to beam, in default its placed 90 degrees to the beam relative angle is added to this angle, takes angle in radians

-relative_angle_deg - allows for giving value in degrees

-mirrored - is a flag which rotates the component by 180 degrees, to be able to easy component placement according to its role, when True it will collimate beam if starting in focus point, when False it will focus beam, In 2D view of line the dot on the end of parabola is the higher value mirror end, where parabola, has higher value.
